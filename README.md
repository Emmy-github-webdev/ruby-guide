## Getting Started

You will need to have Ruby installed.

Clone this repo and enter the directory.

bash
git clone [Ruby Guid](https://github.com/Emmy-github-webdev/ruby-guide).
cd ruby-guide


Run the Interactive Ruby Shell.

bash
irb
